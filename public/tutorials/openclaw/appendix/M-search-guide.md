
# 🔍 搜索功能使用指南

> 本教程的搜索功能专注于标题搜索，帮助你快速找到需要的章节


## 🎯 搜索技巧

### 1. 使用关键词

**推荐做法**：

✅ 飞书
✅ 配置
✅ Skills
✅ 问题
✅ API


**不推荐**：

❌ 如何配置飞书Bot（太长）
❌ 我想知道怎么安装（口语化）


### 2. 中文搜索

支持中文关键词搜索：

```text
飞书 → 找到所有包含"飞书"的标题
配置 → 找到所有包含"配置"的标题
问题 → 找到所有包含"问题"的标题
```

### 3. 英文搜索

支持英文关键词搜索：

```text
API → 找到所有包含"API"的标题
Skills → 找到所有包含"Skills"的标题
Bot → 找到所有包含"Bot"的标题
```

### 4. 组合搜索

使用空格分隔多个关键词：

```text
飞书 配置 → 找到同时包含"飞书"和"配置"的标题
API 问题 → 找到同时包含"API"和"问题"的标题
```

### 5. 模糊匹配

搜索支持前缀匹配：

```text
配 → 可以匹配"配置"
问 → 可以匹配"问题"
Ski → 可以匹配"Skills"
```


## 🚀 高级功能

### 实时搜索

输入2个字符后，搜索会自动开始（延迟300ms），无需点击搜索按钮。

### 结果高亮

搜索结果中，匹配的关键词会用黄色背景高亮显示。

### 分类标签

搜索结果会显示内容分类：

- 📚 文档 - docs目录下的文档
- 📖 附录 - appendix目录下的附录
- 💡 示例 - examples目录下的示例

### URL参数搜索

可以通过URL参数直接搜索：

```yaml
https://your-site.com/search?q=飞书
```
